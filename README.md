# VideoCutCS

WinUI 3 製の軽量動画カットツールです。FFmpeg を内部で使用し、再エンコードなしの高速カット・スマートカット・スナップショット保存などに対応しています。

## ダウンロード

**[最新版はこちら → Releases](https://github.com/Nevaseme/VideoCutCS/releases/latest)**

ZIPを解凍して `VideoCutCS.exe` を実行するだけで使えます。.NETランタイム・Windows App SDK・FFmpeg はすべて同梱済みのため、別途インストール不要です。

> **注意:** Visual C++ 再頒布可能パッケージが必要ですが、Windows 10/11 では通常インストール済みです。

## 機能

### カット

| 機能 | 説明 |
|---|---|
| **高速カット** | 再エンコードなし（ストリームコピー）。フレーム単位の誤差はあるが瞬時に完了 |
| **スマートカット** | 開始点～次のキーフレームのみ再エンコードし、残りはストリームコピーで高速処理。フレーム精度で正確にカット |
| **キーフレーム吸着** | 開始・終了位置を最寄りのキーフレームへ自動スナップ（高速カットと組み合わせて誤差ゼロ） |
| **バッチカット** | 複数セグメントを登録し、一括カット＆結合して1ファイルに出力 |
| **HWエンコード** | スマートカット時に NVENC / QSV / AMF を自動検出して利用（対応GPU搭載時） |

### タイムライン

| 機能 | 説明 |
|---|---|
| **ズーム** | 1x ～ 100x。スライダー・テキスト入力・Shift+ホイールで操作 |
| **ドラッグシーク** | タイムラインをドラッグして再生位置を変更（50ms スロットリングでラグ低減） |
| **ホバーツールチップ** | マウス位置に対応するタイムスタンプをミリ秒精度で表示 |
| **選択範囲表示** | 開始～終了の範囲をタイムライン上に黄色でハイライト |
| **セグメント表示** | 登録済みセグメントをタイムライン上に黄色矩形で可視化。ホバー/選択時に強調 |
| **自動追従** | 再生中はプレイヘッドを中心にスクロールビューが自動追従 |

### 再生・操作

| 機能 | 説明 |
|---|---|
| **再生速度** | 0.1x ～ 10.0x。スライダー・テキスト入力・ホイールで操作 |
| **マウスホイールシーク** | ズームレベルに応じてステップを自動調整（低倍率: 10秒 → 中倍率: 1秒 → 高倍率: 0.1秒） |
| **キーフレームジャンプ** | 動画読み込み時にキーフレーム位置を自動解析し、前後のキーフレームへ瞬時に移動 |
| **スナップショット** | 現在フレームをPNGで保存 |
| **時間直接入力** | 現在位置・開始時間・終了時間のテキストボックスに時間を直接入力して移動・設定可能 |
| **ドラッグ&ドロップ** | ファイルをウィンドウにドロップして開く |

### セグメントリスト

| 機能 | 説明 |
|---|---|
| **セグメント追加** | 現在の開始/終了範囲をセグメントとして登録 |
| **セグメント削除** | 選択中のセグメントを削除（Delete キー対応） |
| **バッチカット実行** | 登録済みの全セグメントを一括でカット・結合して出力 |
| **タイムライン連動** | セグメントにホバー/選択するとタイムライン上の対応矩形がハイライト |

### 出力形式

| 用途 | 形式 |
|---|---|
| カット | MP4 / MPEG-TS / MOV |
| スナップショット | PNG |

## キーボードショートカット

| キー | 動作 |
|---|---|
| `Space` | 再生 / 一時停止 |
| `←` / `→` | 5秒シーク |
| `Ctrl + ←` / `Ctrl + →` | 1秒シーク |
| `Alt + ←` / `Alt + →` | 前後のキーフレームへジャンプ |
| `I` | 開始点を現在位置にセット |
| `F` | 終了点を現在位置にセット |
| `S` | スナップショット保存 |
| `Delete` | 選択中のセグメントを削除 |
| ホイール | シーク（ズームレベルに応じたステップ） |
| `Shift + ホイール` | タイムラインズーム |

## ビルド方法（開発者向け）

### 必要なもの

- Visual Studio 2022 または .NET 8 SDK
- Windows App SDK ワークロード
- `Executables/` フォルダに `ffmpeg.exe` と `ffprobe.exe` を配置

```bash
git clone https://github.com/Nevaseme/VideoCutCS.git
cd VideoCutCS

# ffmpeg を Executables/ に手動配置後
dotnet build
dotnet publish -c Release -r win-x64 --self-contained -o publish/
```

### テスト実行

```bash
dotnet test VideoCutCS.Tests/
```
